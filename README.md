# Student Placement & Career Success Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using a Student Placement & Career Success dataset containing 25,000 student records.

The dashboard analyzes placement performance, salary trends, CGPA impact, AI skill adoption, burnout analysis, and branch-wise career success.

---

## Tools & Technologies Used

* Power BI
* Power Query
* DAX
* CSV Dataset
* Data Visualization

---

## Dashboard Features

* Placement Percentage Analysis
* Average Salary Analysis
* Branch-wise CGPA Comparison
* DSA Problem Solving Insights
* Burnout & Aptitude Analysis
* Student Degree Distribution
* Placement vs CGPA Correlation

---

## Key Insights

* Students with higher CGPA received better salary packages.
* AI & DSA skills positively impacted placements.
* Burnout scores affected student performance.
* Placement rates varied significantly across branches.
